## Laither 叠时

**Laither 叠时** 是王磊的个人工作室，2026 年于南京市浦口区成立。

"叠时"来自"每一秒都在叠加前一秒的成果"——把日常里微小的自动化、工具化积累叠起来，就是工作室想做的事。

## 做什么

- **Web/SaaS**：`stock.laither.com`（投资复盘）· `math.laither.com`（少儿竞赛备考）
- **iOS App**（2026 年底 Phase 3）：Flutter 跨端 + SwiftUI 原生
- **开源工具**：Android 任务提醒 App / 跨设备归档工具 / iPhone 照片扫描

## 关于我

王磊，全栈工程师，日常写 Java / C++ / Kotlin / Python。主业做手机车机互联（投屏、蓝牙、Wi-Fi 子系统），业余做个人产品。
